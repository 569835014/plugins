## 常用命令说明

``` bash

# 编译生成umd格式的`we-plugins.js`库文件
npm run build

# 尝试用`rollup`进行库文件的构建（*暂不可用*）
npm run rollup


# 提供开发文档网站本地开发环境，支持热更新、ES6、Less
npm run docs:dev

# 编译出供发布到生产环境用的开发文档网站。编译后的文件会存放在`dist`目录下
npm run docs:build

# 将dist目录下的文件（即开发文档网站静态内容）发布到`gh-pages`分支上
npm run docs:publish

```
