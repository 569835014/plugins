# 介绍

----

`WePlugins` 是一个项目插件库，集成平时开发用到的很多插件和工具库

## 特性

- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
- [Electron](http://electron.atom.io/)
- [NW.js](http://nwjs.io)
